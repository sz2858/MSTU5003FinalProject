Machines Connected with Computer Programming

Cell Phones
Programs allow you to make a phone call or send text messages. When you search for a contact, a program finds the correct phone number.
Computer Software
Everything a computer does, from browsing the Internet to writing documents or playing the music, works because of code written by a computer programmer.
Games
Consoles are just another type of computer, and all the games that run on them are programs.
Washing Machines
Washing machines are programmed to follow different cycles. Computer code controls how hot the water is and how long the wash takes.
Cars
In some cars, computer programs monitor the speed, temperature, the amount of fuel in the tank, and even help to control the brakes.

How Computer Programs Work

Although computers might seem very smart, but they are actually just boxes that follow instructions very quickly and accurately.
Computers can't think
A computer can do nothing by itself. Computer programmer is the most important element in programming as they give out instructions.
Write a program
You can tell a computer what to do by writing a set of very detailed instructions called a program. The instructions should be small enough for the computer to understand.

Programming Languages

Computers can only follow instructions in a language they understand. It's up to the programmer to choose which languaeg is best for the task.
C
A powerful language for building computer operating system.
Ada
Used to control spacecraft, satellites, and airplanes.
Java
Works on computers, cell phones, and tablets.
Matlab
Ideal for programs that need to carry out lots of calculations.
Ruby
Automatically turns lots of information into web pages.
